## Ficheiros
- `index.html` — App principal (Finance v10)
- `apps-script.js` — Backend Google Apps Script
- `apple-touch-icon.png` — Ícone do app para iOS

## Instalar no iPad
1. Abre o link no Safari
2. Partilhar → Adicionar ao Ecrã de Início

## Configurar Cloud Sync
1. Vai a script.google.com → Novo projecto
2. Cola o conteúdo de `apps-script.js`
3. Deploy → Web App → Anyone
4. Copia o URL `/exec`
5. No app → Settings → Cloud Sync → cola o URL
